# Four-Week Master Roadmap

## Deadline contract

The final submission is due August 15, 2026. Each week produces four complete independent deliverables. Work during a week uses only the frozen beginning-of-week repository and fixtures, never another member's work-in-progress.

## Scope contract

Required: positive-control reproduction, one licensed domain, one 124M–160M screening model, one primary nonzero budget, random/schedule-only/selection-only/joint policies, paired chains, held-out NLL, tail retention, exact accounting, complete paper, and validity audit.

Deferred: second domain, 410M/1B/7B scales, independent model family, oracle, broad downstream suite, universal or optimality claims.
